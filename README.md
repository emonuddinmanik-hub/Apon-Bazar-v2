
# 🍯 Apon Bazar

Premium Organic Honey Website built with HTML, CSS & JavaScript.

Welcome to **Apon Bazar**, a modern and responsive honey showcase website designed to promote premium organic honey products with a clean, elegant, and user-friendly experience.

This project combines beautiful design, smooth navigation, and a professional product presentation system to help honey businesses build trust and increase customer engagement online.

---

## 🌟 Project Overview

Apon Bazar is a fully responsive landing page and product showcase website created for organic honey brands. The website highlights premium honey products, special offers, combo packages, and customer-friendly ordering options through a visually appealing interface.

Whether viewed on mobile, tablet, or desktop, the website provides a seamless browsing experience for customers.
